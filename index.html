Cardinal is a small “mobile first” CSS framework for front-end developers who build responsive web applications.

Menu Back to top

Get started

This framework includes a minimal file structure and useful CSS styles to kickstart your next responsive web project.

Cardinal focuses on performance, good typography, and accessibility first. It is a boilerplate that is ready for you to expand and make your own.

The essentials are all here, but the creativity is up to you.
