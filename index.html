Cardinal Front-end Framework

by Chris Bracco

A barebones collection of killer front-end code for starting web projects. Includes some opinionated default styles, a typographic baseline grid, a fluid grid system inspired by Proportional Grids, and lots of code from HTML5 Boilerplate, normalize.css, jQuery, Modernizr, and more.

Many thanks to all of the amazing people who helped build the above.
